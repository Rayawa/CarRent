## 二、实施计划（按优先级分组）

### 第 1 批：高优先级 — 票据生成 + 用户名登录 + 分类查询 + 分组统计

这些是直接影响评分的关键缺失功能。

#### 1.1 票据生成 (功能 5)
- **文件**: `car_rent.cpp` — 新增 `printRentTicket(RentRecord&)` 和 `printReturnTicket(RentRecord&, double, double)` 函数
- **实现**:
  1. 租车票据：带 `===== 汽车租赁公司租车票据 =====` 抬头，含车辆信息（品牌/车牌/型号）、用户信息（姓名/驾照号）、租车日期、预计归还日期、日租金、押金金额，分隔线美化
  2. 退车收据：带 `===== 汽车租赁公司租车费用收据 =====` 抬头，含租车天数、日租金、总费用、押金、退款金额、实际归还日期
- **调用**: 在 `menu3Rent()` 租车成功后调用 `printRentTicket()`，在 `menu4Return()` 退车成功后调用 `printReturnTicket()`

#### 1.2 用户名登录 (功能 10 补全)
- **文件**: `car_rent.cpp`
- **实现**:
  1. 新增 `USERNAME_MAX_LEN` 常量 (32)
  2. 新增 `usernameStore[32]` 全局变量和 `usernameReady` 标志
  3. 新增 `saveUsername()` / `loadUsername()` 持久化函数
  4. 修改 `login()`：先输入用户名，再输入密码（均最多 3 次尝试）
  5. 首次运行时同时设置用户名和密码
  6. 修改密码时保留用户名不变

#### 1.3 分类查询 (功能 7 补全)
- **文件**: `menu5_query.cpp` — 完全重写 `menu5Query()`
- **实现**:
  1. 车辆查询子菜单：按类型/品牌/颜色/状态分别筛选
  2. 用户查询子菜单：按姓名/驾照号/性别/驾龄范围筛选
  3. 租车查询子菜单：按日期范围/状态筛选
  4. 每种查询独立循环，结果以表格形式输出

#### 1.4 分组统计 (功能 8 补全)
- **文件**: `menu6_statistics.cpp` / `car_rent.cpp` — 新增统计函数和重写菜单
- **实现**:
  1. `countVehiclesByBrand()`、`countVehiclesByType()`、`countVehiclesByColor()` — 分组计数
  2. `countRentersByGender()`、`countRentersByDrivingYears()` — 用户分组
  3. `countRentsByMonth()` — 按月统计租车量
  4. 重写 `menu6Statistics()` 展示分组统计结果

以上 4 项估算代码量：约 300-400 行新增代码。

### 第 2 批：中优先级 — 代码规范完善 + 重名检查 + 分页

#### 2.1 文件头部块注释
- **所有 10 个 .cpp 文件** — 添加标准头部：
```cpp
/*
 * 汽车租赁管理系统
 * 文件: xxx.cpp
 * 功能: 车辆信息管理子菜单
 * 作者: [姓名]
 * 日期: 2026-07
 */
```

#### 2.2 函数文档注释
- **car_rent.cpp 中约 66 个函数** — 为每个公开函数添加：
```cpp
/*
 * 功能: 按ID查找车辆
 * 入参: id - 车辆编号
 * 返回: 找到返回Vehicle指针，未找到返回nullptr
 */
```

#### 2.3 命名规范化
- `isLeap` → `isLeapYear`
- `validDate` → `isValidDate`
- `diffDays` → `calcDateDiff`
- `trimNewline` → `trimNewLine`
- `nowDate` → `getNowDate`，`nowDateTime` → `getNowDateTime`

#### 2.4 重复数据提醒
- **文件**: `car_rent.cpp`
- 新增 `isPlateNoDuplicate(plateNo)`、`isLicenseNoDuplicate(licenseNo)`、`isIdCardDuplicate(idCard)` 函数
- 在 `addVehicle()` 和 `addRenter()` 中调用检查，返回错误提示

#### 2.5 分页查询
- **文件**: `car_rent.cpp` — 新增 `printWithPagination(printFunc, totalCount, pageSize)` 通用函数
- 改造 `printAllVehicles`/`printAllRenters`/`printAllRents` 支持分页

### 第 3 批：低优先级 — 链表重构 + 扩展功能

#### 3.1 链表实现 (功能 3 的必要条件)
- **文件**: 新增 `linked_list.h` / `linked_list.cpp`

- 将车辆/用户/租车记录中**至少 3 个功能**改用链表管理

- 错误分类提示：区分"为空"/"超出范围"/"格式错误"

## 四、风险与约束

1. **保持现有 can_rent.cpp include 架构不变** — 不引入 .h 文件，保持单文件编译模式（除链表模块外）
2. **链表实现需兼容现有数组接口** — 链表改造后，现有 `printAll` / `find*` 等函数需适配
3. **二进制文件格式变更需谨慎** — 数据结构修改（如添加用户名字段）可能导致旧数据文件不兼容
4. **遵循指导书评分标准** — 票据布局、统计维度等需符合指导书描述
5. **函数长度控制在 80 行内**
