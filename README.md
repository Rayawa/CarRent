# CarRent - 汽车租赁管理系统

## 项目简介

CarRent 是一个基于 C++ 开发的命令行汽车租赁管理系统，实现车辆信息管理、租车用户管理、租车/退车业务办理、信息查询、统计汇总、报表导出等核心功能。系统采用静态数组存储数据，结合二进制文件持久化，提供稳定高效的数据操作能力。

- **开发语言**: C++11
- **开发环境**: 跨平台（Windows / macOS / Linux）
- **构建工具**: Make / CMake
- **IDE**: 兼容 CLion / VS Code（已配置 .clangd）

### 源文件架构说明

程序通过 `#include` 方式组装：`main.cpp` → `car_rent.cpp` → `menu1~9.cpp`。

| 文件 | 职责 |
|------|------|
| **car_rent.cpp（核心）** | 定义全部常量、枚举、结构体、全局数组，提供输入、日期、加密、持久化、增删改查等公共方法，并包含主菜单入口 |
| **menu1~9.cpp** | 各功能模块的交互子菜单，每个文件仅包含一个 `menuXxx()` 函数 |

`car_rent.cpp` 内部按功能划分为 **16 个部分**：定义与常量 → 系统工具 → 输入方法 → 时间日期 → 状态映射 → 结构体初始化 → ID 分配与查找 → 日志与持久化 → 密码管理 → 数据增删改 → 打印信息 → 统计汇总 → 综合输出 → 子菜单引用 → 界面渲染与程序入口。

## 功能模块

| 序号 | 菜单 | 功能说明 |
|------|------|----------|
| 1 | 车辆信息管理 | 添加/删除/修改/显示全部车辆（车牌号、品牌、类型、颜色、购买日期、保险、日租金、里程） |
| 2 | 租车用户信息管理 | 添加/删除/修改/显示全部用户（姓名、年龄、性别、电话、驾照号、身份证号、驾龄） |
| 3 | 租车业务办理 | 选择车辆和用户，录入日期，系统自动计算押金（日租金 × 3）并创建租车记录 |
| 4 | 退车业务办理 | 按实际归还日期计算租车天数和总费用，自动退还押金差额，支持打印租车票据 |
| 5 | 信息查询 | 按关键词模糊搜索车辆（品牌/车牌）、用户（姓名/驾照）、租车记录（ID） |
| 6 | 统计汇总 | 展示车辆总数、可租/已租数量、用户总数、租车记录数、进行中记录、总收入，按品牌统计 |
| 7 | 输出全部信息 | 一次性列表展示所有车辆、用户、租车记录 |
| 8 | 系统设置 | 修改密码、导出报表 TXT、查看操作日志、批量清空测试数据 |
| 9 | 扩展功能 | 跨实体模糊搜索、车辆状态汇总（可租/已租/维修） |
| 0 | 退出系统 | 自动保存全部数据后安全退出 |

## 核心算法说明

| 算法 | 位置 | 说明 |
|------|------|------|
| 日期合法性校验 | `validDate()` | 校验 YYYY-MM-DD 格式，闰年识别，每月天数验证 |
| 日期差计算 | `dateToDays()` / `diffDays()` | 将日期转换为从 2000-01-01 起的天数，用于计算租车天数 |
| 密码加密 | `xorPassword()` | 明文与 0x5A 做异或加密后存储，memcmp 校验 |
| ID 分配 | `nextXxxId()` | 遍历现有记录取最大 ID + 1，确保 ID 唯一且递增 |
| 模糊搜索 | `strstr()` 子串匹配 | 在品牌/车牌/姓名/驾照号等字段中做关键词匹配 |
| 退车费用计算 | `returnRent()` | 天数 × 日租金 = 总费用，押金 - 总费用 = 退款 |
| 数据持久化 | `saveXxx()` / `loadXxx()` | 二进制文件格式：[count(4字节)] + [记录块]，加载时含合法性校验 |

## 代码规范

- 常量使用 `constexpr`，避免魔法数字
- 函数单一职责，每个函数附带中文注释说明
- 字符串转换使用 `strtol` / `strtod` 替代 `atoi` / `atof`，避免未检测的转换错误
- 窄化转换使用 `static_cast` 显式声明，消除编译器警告
- 关键业务逻辑均添加算法注释
- 文件内按功能分组，每组以 `//==========================第X部分：XXX====================` 标识

## 构建与运行

### 方式一：使用 Make

```bash
make              # 编译
./CarRent         # 运行
make clean        # 清理编译产物
make distclean    # 清理编译产物 + 数据文件
```

### 方式二：使用 CMake（CLion 推荐）

在 CLion 中直接打开项目目录，CMake 会自动配置。或在终端中：

```bash
mkdir build && cd build
cmake ..
make
./CarRent
```

### 方式三：使用 g++ 直接编译

```bash
g++ -std=c++11 -Wall -Wextra -o CarRent src/main.cpp
./CarRent
```

## 使用说明

1. **首次运行**：系统提示设置登录密码（6~16 位），密码经 XOR 加密后存入 `pwd.dat`
2. **登录**：输入正确的密码（最多 3 次尝试机会）
3. **主菜单**：选择 1~9 进入各功能子菜单，0 退出
4. **车辆管理**：先添加车辆，再办理租车
5. **用户管理**：添加租车用户信息后方可办理租车
6. **租车办理**：选择车辆 ID 和用户 ID，录入日期后自动生成租车记录
7. **退车办理**：输入记录 ID 和实际归还日期，自动计算费用和退款
8. **查询统计**：支持关键词模糊查询和多维度统计
9. **系统设置**：修改密码、导出报表、查看日志、清空数据
